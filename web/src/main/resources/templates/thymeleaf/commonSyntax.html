1. 表达式语法之运算符

常量表达式

  • 字符串常量 th:text="'hello, word'" -

  • 数字常量 th:text="'2019, 11'" -

  • 布尔值常量 th:text="${result}" -

  • 空值常量判断 th:text="${result == null}" -

字符串拼接

  • 一: th:text="${#strings.append('luna','op')}" -

  • 二: th:text="${#strings.concat('luna','op')}" -

  • 三: th:text="${'luna' + 'ob'}" -

  • 四: th:text="'luna' + 'ob'" -

  • 五: th:text="luna + ob" -

  • 六: th:text="|luna| + |oq|" -

算术运算符

  • 一:3 + 2= th:text="${3 + 2}" -

  • 二:3 - 2= th:text="${3 - 2}" -

  • 三:3 * 2= th:text="${3 * 2}" -

  • 四:3 / 2= th:text="${3 / 2}" -

  • 五:3 & 2= th:text="${3 % 2}" -

关系运算符

  • 一:1 > 1 = th:text="${1 gt 1}" -

  • 二:1 < 1 = th:text="${1 lt 1}" -

  • 三:1 >= 1 = th:text="${1 ge 1}" -

  • 四:1 <= 1 = th:text="${1 le 1}" -

  • 五:1 == 1 = th:text="${1 eq 1}" -

  • 六:1 != 1 = th:text="${1 ne 1}" -

三目运算符

th:text="${1 != 1 } ? |条件成立| : |条件不成立| "
